学長 木下 勇人
学長からのご挨拶
木下 勇人
税理士AI大学 学長

私は長年、相続専門の税理士として現場に立ちながら、「業界全体の生産性と価値を高めることで、中小企業をより深く支えることはできないか」という問いを抱いてまいりました。

税理士事務所は、中小企業にとって"最後の拠り所"とも言える存在です。しかし、日々の業務に追われ、本来向き合うべき支援に十分な時間を割けていない現実があります。

当初、私はAIによって業務を効率化し、時間を取り戻すことで、本来の役割に集中できる環境を実現したいと考えていました。実際に、記帳やチェック、資料作成といった業務は着実に軽減され、AIは現場において価値発揮を後押しする存在として活躍しています。

その一方で、AIの進化は、もはや私の想定をはるかに超える領域に入ってきました。いまやAIは、単なる効率化の道具にとどまらず、税理士業務そのものの一部を担い得る段階に入りつつあります。これは大きな可能性であると同時に、これまでの延長線上にいるだけでは立ち行かなくなるという現実も示しています。

だからこそ今、私たちは業務を効率化するだけでなく、自らの役割そのものを見つめ直し、仕事のあり方を変えていく必要があります。AIに任せられる業務はAIに任せる。そのうえで私たちは、人にしかできない領域へと軸足を移していく。お客様の未来に向き合い、経営課題に寄り添い、事業の成長を支える——そうした本質的な役割に、これまで以上に力を注いでいくことが重要だと考えています。

AIの進化を前に、不安や戸惑いを感じている方もいらっしゃるかもしれません。私自身、決してITに精通していたわけではありません。だからこそ、誰もが現場で無理なくAIを活用し、この変化に対応できる環境が必要だと考え、本サロンを立ち上げました。本サロンでは、やさしく、現場目線で、実務にまで落とし込むことを大切にしています。同じ志を持つ仲間とともに、実践を重ねながら成長していける場でありたいと考えています。

税理士一人ひとりがAIを活用し、提供できる価値を高めていく。その積み重ねが中小企業を支え、企業の成長につながり、やがて社会全体を前へと動かしていく——私はそう信じています。

変化は、すでに始まっています。その変化をどう捉え、どう行動するかは、私たち自身に委ねられています。

AIに対する不安があっても問題ありません。その不安を抱えたままでも、一歩を踏み出すことにこそ意味があります。
ぜひ、新たな仲間として、ともにこの変化を前向きに捉え、未来を切り拓いていきましょう。
皆さまのご参加を、心よりお待ちしております。

FAQ

よくあるご質問

Q
AIの知識がまったくなくても大丈夫ですか？
+
はい、まったく問題ありません。このスクールは「AIをまだ触ったことがない」という方のためにこそ設計されています。理論の説明はなく、当日一緒に触れていただく形式なので、知識ゼロからでも安心してご参加いただけます。
Q
スマホだけでも参加できますか？
+
スマホでのご参加も可能です。ただ、実際に業務でAIを活用する際はパソコンがあるとより便利です。まずはスマホで第1回を体験してみてください。
Q
当日参加できなかった場合はどうなりますか？
+
各回のアーカイブ動画をご提供しますので、ご都合のよいタイミングでご視聴いただけます。ただ、ライブ参加の方がリアルタイムで質問できるため、できるだけライブでのご参加をお勧めしています。
Q
無料体験後、必ず入会しなければなりませんか？
+
いいえ、無料体験後にご入会を強制することは一切ありません。まずは第1回を体験していただき、ご自身に合うかどうかをご確認ください。
Q
税理士以外（会計士・社労士など）でも参加できますか？
+
税理士・公認会計士の先生方を主な対象としておりますが、士業・会計事務所スタッフの方もご参加いただいております。ご不明な点はお気軽にお問い合わせください。
Q
紹介割引を受けるにはどうすればよいですか？
+
ご紹介者様から招待コードをお受け取りください。コードをお持ちでない場合や不明な点がある場合は、税理士AI大学 事務局までお気軽にお問い合わせください。紹介割引が適用されると、通常月額50,000円が30,000円でご入会いただけます。
Q
解約はいつでもできますか？
+
はい、解約はいつでも可能です。翌月の講座引き落とし日の10日前までにご連絡いただければ、その月末での解約となります。なお、入会中は紹介割引価格が継続して適用されますが、再入会の際は通常料金（月額50,000円）となりますのでご注意ください。
Q
顧問先の情報をAIに入力しても大丈夫ですか？
+
AIツールをご利用の際は、顧客名や個人を特定できる情報は入力せず、数値データや一般的な取引内容のみをご使用いただくことを推奨しております。セキュリティポリシーや具体的な活用方法については、入会後のスターターガイドおよびSlackコミュニティにて詳しくご案内しています。
